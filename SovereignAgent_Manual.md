# Sovereign Agent: Complete Operations Manual

## 1. Overview and Manifesto
In the contemporary landscape of automated HR filters and "Ghost Jobs," the job seeker requires a technological advantage. Sovereign Agent serves as a highly intelligent, localized automation engine designed to handle the repetitive aspects of the job application process, allowing the user to focus on interview preparation and career strategy.

## 2. Security and Privacy Policy
Privacy is a fundamental component of the Sovereign Agent architecture.
* **Localized Storage**: All resumes, AI configurations, and authentication credentials are stored exclusively on the user's local machine.
* **No Telemetry**: The system does not collect data or communicate with external servers beyond the specified AI providers and job platforms.
* **Auditability**: As an open-source project, the codebase is fully transparent and available for security auditing.

## 3. System Requirements and Installation

### Windows Installation
Windows users may choose between a portable executable or running from the source code.

#### Option A: Portable Executable (Recommended)
1. Download the latest `SovereignAgent_Portable.zip` from the official repository releases.
2. Extract the contents to a dedicated directory (e.g., `C:\SovereignAgent`).
3. Execute `SovereignAgent_Portable.exe` to launch the application.

#### Option B: Installation from Source
1. Install Git from git-scm.com.
2. Install Python 3.10+ from Python.org, ensuring the "Add Python to PATH" option is selected.
3. Open a terminal and execute the following commands:
   ```cmd
   git clone https://github.com/wisdomgreat/JobAutomation
   cd JobAutomation
   python -m venv venv
   venv\Scripts\activate
   pip install -r requirements.txt
   python gui.py
   ```

### macOS Installation
macOS requires execution from the source code via the Terminal.
1. Install Homebrew:
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
2. Install dependencies:
   ```bash
   brew install git python3
   ```
3. Execute the setup:
   ```bash
   git clone https://github.com/wisdomgreat/JobAutomation
   cd JobAutomation
   python3 -m venv venv
   source venv/bin/activate
   pip3 install -r requirements.txt
   python3 gui.py
   ```

### Linux and Unix Installation
1. Install system dependencies:
   ```bash
   sudo apt update
   sudo apt install git python3 python3-pip python3-venv xvfb
   ```
2. Execute the setup:
   ```bash
   git clone https://github.com/wisdomgreat/JobAutomation

   python3 -m venv venv
   ```

## 4. Post-Installation Configuration

### AI Provider Configuration
The application requires an external Large Language Model (LLM) to perform analysis and document tailoring.
1. Navigate to the **System Core** tab via the sidebar.
2. Select an AI Provider (e.g., OpenAI, Anthropic, or OpenRouter).
3. Select the specific model (e.g., gpt-4o-mini).
4. Enter the authorized API Key and select **SAVE KEY**.
5. Use the **Test AI Synapse** button to verify the connection.

### Identity Profile Management
1. In the **System Core** tab, select **EDIT GLOBAL IDENTITY**.
2. Complete the **Identity Commander** form with professional history, job preferences, and demographics.
3. Select **SAVE IDENTITY** to commit changes.

### Resume Asset Management
1. Navigate to the **Asset Hub** tab.
2. In the **Master Assets** section, select **Upload Resume**.
3. Provide your primary resume in PDF or DOCX format. This file serves as the basis for all AI-generated tailoring.

## 5. Operational Procedures

### Automated Job Scanning
1. Navigate to the **Target Scan** tab.
2. Define the search parameters in the **Search Matrix** (Job Title and Location).
3. Select the target platforms (e.g., LinkedIn, Indeed).
4. Select **RUN FULL AUTO-PIPELINE**.
5. Progress can be monitored via the **Dashboard** and **Live Console**.

### Candidate CRM and Tracking
1. Navigate to the **Candidate CRM** tab to view discovered opportunities.
2. Review the **Match Score** generated by the AI for each entry.
3. Select **Quick Apply** to generate tailored documents immediately. These will be placed in the `output/` directory.
4. Mark entries as **Applied** using the checkmark icon to maintain an organized queue.

### Targeted Surgical Strike
For specific job opportunities found outside the automated scan:
1. Navigate to the **Target Scan** tab.
2. Input the direct job URL into the **Surgical Precision Strike** field.
3. Select **GENERATE NEW KIT**.
4. Retrieve the tailored assets from the `output/` directory.

## 6. Advanced Configuration: Email Integration
The Email Scanner requires an **App Password** for secure access to job alert notifications.

### Google / Gmail Setup
1. Access Google Account Security settings and enable 2-Step Verification.
2. Navigate to the **App passwords** section.
3. Create a password labeled "Sovereign Agent."
4. Input the provided 16-character code into the application's **Security** tab.

### Yahoo and Microsoft Setup
Similar "App Password" or "Advanced Security" sections must be used to generate a unique token for the application, ensuring your primary password remains private.

## 7. Development and Distribution
To build a standalone executable for distribution:
1. Execute the build script:
   ```bash
   python build.py
   ```
2. The bundled assets will be generated in the `dist/` directory.

## 8. Troubleshooting and Support
* **Terminal Errors**: Ensure the virtual environment is active before running commands.
* **Browser Failures**: Verify that a modern browser (Chrome or Edge) is installed and updated.
* **UI Instability**: On Windows, ensure you are running the latest version of the application to include stability patches for the GUI engine.

For additional support or to report bugs, please utilize the official repository issues tracker.
