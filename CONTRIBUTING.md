# 🤝 Contributing to Sovereign Agent

Welcome to the movement! We are building a future where job seeking is fair, efficient, and powered by intelligent assistance. We love contributions from the community.

### 🚀 How to Contribute

1.  **Report Bugs**: If you find something broken, search existing issues first. if not found, open a new one with clear reproduction steps.
2.  **Suggest Features**: Have an idea for a better "Stealth" mechanism or a new platform? Let's talk in the issues!
3.  **Submit Pull Requests**:
    - Fork the repository.
    - Create a feature branch (`git checkout -b feature/amazing-feature`).
    - Commit your changes (`git commit -m 'Add some amazing feature'`).
    - Push to the branch (`git push origin feature/amazing-feature`).
    - Open a Pull Request.

### 🎨 Design Philosophy
- **Privacy First**: We never track user data.
- **Stealth Always**: Automation must be indistinguishable from human activity.
- **Executive Polish**: The UI and output should always feel high-end.
