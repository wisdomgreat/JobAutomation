# Sovereign Agent

## Introduction
Sovereign Agent is an advanced automation system designed to streamline the employment application process. By utilizing localized intelligence, the system automates job discovery, calculates candidate alignment, and generates tailored application assets to assist job seekers in navigating modern recruitment filters.

## Quick Start
For users on Windows, the fastest way to begin is by using the Portable Executable:
1. Download the latest `SovereignAgent_Portable.zip` from the [Releases](https://github.com/wisdomgreat/JobAutomation/releases) page.
2. Extract and run `SovereignAgent_Portable.exe`.

## Documentation
Comprehensive instructions regarding installation, configuration, and operational procedures are available in the **[Sovereign Agent Operations Manual](SovereignAgent_Manual.md)**.

## Security and Privacy
The Sovereign Agent architecture prioritizes user data integrity:
* All application data and credentials remain exclusively on the user's local hardware.
* No telemetry or personal data collection is performed by the system.
* The system is open-source and subject to public audit.
