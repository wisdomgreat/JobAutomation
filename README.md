## 💻 Technical Setup (For Developers)

If you prefer to run from source or contribute to the mission:

### 1. Prerequisites
- **Python 3.10+**: [Download](https://www.python.org/downloads/) (Ensure "Add Python to PATH" is checked).
- **Git**: [Download](https://git-scm.com/downloads).

### 2. Manual Installation
For full setup walkthroughs tailored to **Windows**, **macOS**, and **Linux/Unix**, please read the **[Multi-Platform Setup Guide](SETUP_GUIDE.md)**!

```bash
# General Quick-Run (Ensure Python 3.10+ is installed)
git clone https://github.com/wisdomgreat/JobAutomation
cd JobAutomation

# Create and activate environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install gear & Launch
pip install -r requirements.txt
python gui.py
```
